# 釐清問題

若來源日期包含某部電影的場次，但該電影在「目標日期」時已經下映（或尚未上映），複製操作應如何處理這些無效場次？

# 定位

Feature：複製時刻表.feature
Constraint：MovieShowTime.show_date 必須在 [Movie.release_date, Movie.end_date] 範圍內

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 部分複製：自動略過這些無效的場次，僅複製符合檔期的場次，並提示使用者 |
| B | 全部阻擋：只要由任何一部電影檔期不符，整個複製操作失敗並報錯 |
| Short | Format: Short answer (<=5 words) |

# 影響範圍

- Feature: 複製時刻表
- Rule: 資料驗證與部分成功處理機制

# 優先級

Medium

---
# 解決記錄

- **回答**：A - 部分複製：自動略過這些無效的場次，僅複製符合檔期的場次，並提示使用者
- **更新的規格檔**：spec/features/複製時刻表.feature
- **變更內容**：新增 Rule，規範當電影已下映時，系統會自動略過該場次並執行部分複製
