# 釐清問題

# 定位

Feature：訂票.feature
Rule：目前僅提及「票券使用 QR Code」，未提及通知機制

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 僅站內查看，不需發送外部通知 |
| B | 需發送 Email 通知（包含訂單摘要與連結） |
| C | 需發送簡訊通知（包含取票代碼或連結） |
| Short | Format: Short answer (<=5 words) |

# 影響範圍

- Feature: 訂票 (成功後的後續動作)
- Infrastructure: 需配置 Email 或 SMS 發送服務

Low
