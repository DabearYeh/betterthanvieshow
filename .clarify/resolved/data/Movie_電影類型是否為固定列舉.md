# 釐清問題

`Movie.genre` (影片類型) 的值域是否受限於 Note 中列出的選項（動作、愛情、冒險...），還是允許輸入任意字串？

# 定位

ERM：Movie.genre
屬性：genre

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 嚴格限制，僅能為 Note 中列出的固定選項（需定義 Enum 或 Valid List） |
| B | 開放字串，允許管理員輸入任意類型 |
| C | 混合模式，提供常用選項選單，但也允許手動輸入新類型 |
| Short | Format: Short answer (<=5 words) |

# 影響範圍

- Entity: Movie
- Feature: 建立電影, 編輯電影 (前端輸入介面設計)

# 優先級

Low

---
# 解決記錄

- **回答**：A - 嚴格限制，僅能為：動作、愛情、冒險、懸疑、恐怖、科幻、日本動漫、喜劇
- **更新的規格檔**：spec/erm.dbml
- **變更內容**：更新 Movie.genre 的 Note，將「可選」改為「嚴格限制選項」
