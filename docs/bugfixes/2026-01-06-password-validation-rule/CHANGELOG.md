# 密碼驗證規則修改 - 變更日誌

## 修改摘要
將會員註冊密碼驗證規則從 8 碼（需大小寫+數字）簡化為 6 碼（英文字母或數字）

## 修改檔案

### 1. RegisterRequestDto.cs
- **修改類型**：驗證規則調整
- **變更內容**：
  - 修改正則表達式：`^(?=.*[a-z])(?=.*[A-Z])(?=.*\d).{8,}$` → `^[a-zA-Z0-9]{6,}$`
  - 更新錯誤訊息
  - 密碼範例維持：`pass12`

### 2. AuthController.cs
- **修改類型**：API 文檔更新
- **變更內容**：
  - 更新密碼範例：`Password123!` → `pass12`
  - 更新錯誤訊息說明

- ✅ **現有帳號**不受影響，仍可正常登入
