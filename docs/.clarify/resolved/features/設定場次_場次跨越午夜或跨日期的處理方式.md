# 設定場次_場次跨越午夜或跨日期的處理方式

如果場次跨越午夜（例如晚上 23:00-02:00）,end_time 應如何處理？跨日期還是次日？

# 定義

Feature：設定場次
ERM：MovieShowTime.show_datetime, MovieShowTime.end_time

# 多選項

| 選項 | 描述 |
|--------|-------------|
| A | end_time 允許跨日期，end_time 的日期可能 > show_datetime 的日期 |
| B | 完全禁止跨午夜的場次 |
| C | 允許跨午夜，但時間戳中自動調整日期 |
| Short | 其他處理方式 |

# 影響範圍

- MovieShowTime 的欄位資料模型設計
- 場次衝突檢查的邏輯複雜度
- 時間查詢與排序的正確性
- 驗票時間判斷邏輯

# 優先級

Medium

---

**說明**：規格未考慮夜場跨午夜的情況,需要明確如何處理以確保邏輯的正確性。

---

# 解決記錄

- **決議**：A  可以跨日期（夜場）,end_time 自動計算
- **更新規格檔案**：spec/erm.dbml MovieShowTime 欄位
