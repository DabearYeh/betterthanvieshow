# 編輯電影_若電影已有場次修改資訊是否有限制

編輯電影功能中,若電影已有場次,修改電影資訊是否有限制？

# 定義

Feature：編輯電影功能
關聯：MovieShowTime 與 Movie 的關聯

# 多選項

| 選項 | 描述 |
|--------|-------------|
| A | 無限制：可以修改所有電影資訊 |
| B | 部分限制：可修改片名、簡介等,但不可修改時長 |
| C | 嚴格限制：若有場次則不可修改任何資訊 |
| D | 其他（請說明） |

# 影響範圍

- 編輯電影功能的業務規則
- Movie 與 MovieShowTime 的資料一致性
- 已排程場次的影響

# 優先級

Medium
- 影響電影資訊管理的靈活性

---

**說明**：規格中未明確若電影已有場次時,修改電影資訊是否有限制。

---

# 解決記錄

- **決議**：B — 部分限制,可修改片名、簡介等,但不可修改時長
- **理由**：時長影響場次結束時間,已排程場次不應受影響;其他資訊可彈性修改
- **更新規格檔案**：spec/features/編輯電影.feature
