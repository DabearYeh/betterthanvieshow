# 釐清問題

建立影廳功能執行後，座位配置如何處理？

# 定位

Feature：建立影廳功能
關聯 Feature：設定座位配置功能

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 建立影廳時必須同時設定座位配置 |
| B | 建立影廳後，另外執行設定座位配置功能 |
| C | 建立影廳時只設定座位總數，稍後設定配置 |
| Short | 提供其他簡短答案（<=5 字）|
