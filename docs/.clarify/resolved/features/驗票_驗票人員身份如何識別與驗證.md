# 驗票_驗票人員身份如何識別與驗證

TicketValidateLog.validator 欄位中,驗票人員應如何被識別？手動輸入、自動識別（如登入使用者）,還是工作證？

# 定義

ERM：TicketValidateLog.validator
Feature：驗票功能 Rule: 驗證成功後登記為已入場

# 多選項

| 選項 | 描述 |
|--------|-------------|
| A | 自動識別：使用驗票系統登入者的使用者 ID |
| B | 手動輸入：驗票人員輸入工作 ID 或姓名 |
| C | 掃描識別：掃描工作證的身份碼 |
| D | 系統帳號：使用使用者帳號系統管理 |
| Short | 其他識別方式 |

# 影響範圍

- TicketValidateLog 的欄位填充與驗證
- 驗票功能的實作設計
- 驗票人員的帳號管理系統
- 稽核追蹤的完整性
