# 釐清問題

編輯場次時，除了放映時間、電影、影廳外，其他欄位是否可以編輯？

# 定位

Feature：編輯場次.feature

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 僅時間/電影/影廳：只能編輯放映時間、電影、影廳三個欄位 |
| B | 可改日期：可以將場次移動到其他日期（但目標日期必須是 Draft 狀態） |
| C | 全部可改：所有 MovieShowTime 欄位都可編輯（除了 id） |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

- 編輯場次功能的可編輯欄位定義
- 編輯場次 UI 設計
- 跨日期移動場次的邏輯處理

# 優先級

Medium
- 影響編輯場次功能的完整性

---
# 解決記錄

- **回答**：A - 僅時間/電影/影廳：只能編輯放映時間、電影、影廳三個欄位，日期不可變更
- **更新的規格檔**：spec/features/編輯場次.feature
- **變更內容**：重建編輯場次規格，明確可編輯欄位（時間/電影/影廳）與新增規則「場次日期不可變更」
