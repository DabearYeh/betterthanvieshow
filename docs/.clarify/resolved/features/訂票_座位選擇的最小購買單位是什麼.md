# 訂票_座位選擇的最小購買單位是什麼

座位選擇時使用者必須購買至少 1 張票,還是可以有其他最小單位限制？例如是否支援購買半張票？

# 定義

Feature：訂票功能 Rule: 必須選擇座位

# 多選項

| 選項 | 描述 |
|--------|-------------|
| A | 最少 1 張票,無上限 |
| B | 最少 2 張票（成對購買）|
| C | 依座位類型不同（例單人座）|
| Short | 其他最小單位 |

# 影響範圍

- 訂票流程的驗證邏輯
- 座位選擇 UI 的限制設計
- 商業模式的定價策略

# 優先級

Low
