# 刪除影廳_若影廳有已排程的場次是否可以刪除

刪除影廳功能中，若影廳有已排程的場次,是否可以刪除該影廳？

# 定義

Feature：刪除影廳 feature，關於「影廳有已排程的場次時是否可以刪除」

# 多選項

| 選項 | 描述 |
|--------|-------------|
| A | 不可以刪除，必須先刪除相關場次 |
| B | 可以刪除，自動刪除相關場次 |
| C | 可以刪除，但場次狀態改為已取消 |

- 刪除影廳.feature 中的刪除限制規則
