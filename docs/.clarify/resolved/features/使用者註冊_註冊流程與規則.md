# 使用者註冊_註冊流程與規則

使用者如何註冊？需要驗證什麼？

# 定義

Feature：使用者登入登出系統 > 使用者註冊流程

# 多選項

| 選項 | 描述 |
|--------|-------------|
| A | 註冊時需填入姓名、帳號、密碼；系統驗證帳號唯一性，無需 email 驗證 |
| B | 註冊時需填入姓名、帳號、密碼、email；系統驗證帳號唯一性與 email 格式 |
| C | 註冊時需填入姓名、帳號、密碼、email；系統發送驗證郵件，使用者需點擊確認 |
| D | 其他（請說明） |

# 影響範圍

- User 實體的必填屬性
- 註冊功能的前後置條件
- 錯誤訊息規則（帳號重複、密碼太弱等）

# 優先級
