# 瀏覽場次_如何篩選與顯示場次

瀏覽場次功能中,如何篩選與顯示場次？

# 定義

Feature：瀏覽場次功能
Rule：可以查看電影與影廳資訊、可以查看電影的時段資訊

# 多選項

| 選項 | 描述 |
|--------|-------------|
| A | 先選電影，再顯示該電影的所有場次 |
| B | 先選日期，再顯示當日所有電影場次 |
| C | 先選影廳，再顯示該影廳的所有場次 |
| D | 以上都支援，提供多種瀏覽方式 |
| Short | 其他（請簡短答案<=5 字）|

# 影響範圍

- 瀏覽場次功能的操作流程
- 前台 UI 設計
- MovieShowTime 查詢邏輯

# 優先級

Medium
- 影響前台使用者體驗
- 影響場次瀏覽流程設計

---

# 解決記錄

- **決議**：B  電影 + 影廳類型 + 日期 + 時段
- **流程**：電影 → 日期 → 影廳類型（例 2D、4DX、IMAX）→ 時段顯示，每個影廳下列出所有時段（含開始時間）
- **更新規格檔案**：spec/features/瀏覽場次.feature
