# 釐清問題

編輯影廳功能中，修改座位總數時是否需要檢查現有座位配置？

# 定位

Feature：編輯影廳功能
Rule：可以修改座位總數、修改後的座位總數必須大於 0

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 需要檢查，新座位總數不得小於現有座位數量 |
| B | 需要檢查，修改後需同步調整座位配置 |
| C | 不需要檢查，座位總數為獨立屬性 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

- 編輯影廳功能的前置條件
- Theater 與 Seat 的一致性約束
- 座位配置的調整邏輯
- 測試案例設計

# 優先級

Medium
- 影響編輯影廳功能的驗證邏輯
- 影響 Theater 與 Seat 的一致性管理
