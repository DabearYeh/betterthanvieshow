# 設定座位配置_座位的seat_type如何設定

設定座位配置功能中,座位的 seat_type 如何設定？

# 定義

Feature：設定座位配置功能
Rule：可以設定座位類型

# 多選項

| 選項 | 描述 |
|--------|-------------|
| A | 每個座位可獨立設定類型 |
| B | 整排座位套用同一類型 |
| C | 整個影廳只有一種座位類型 |
| Short | 其他（請簡短答案<=5 字）|

# 影響範圍

- 設定座位配置功能的操作流程
- Seat 實體的 seat_type 屬性管理
- 票價計算邏輯（若不同類型價格不同）
- 座位配置的靈活度

# 優先級

Medium
- 影響座位配置功能的設計
- 影響座位類型管理邏輯
