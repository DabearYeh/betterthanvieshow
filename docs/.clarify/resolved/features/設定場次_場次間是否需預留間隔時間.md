# 釐清問題

設定場次時，場次之間是否需要預留間隔時間（如清場時間）？

# 定位

Feature：設定場次.feature - Rule: 同一影廳同一日期同一時間只能有一個場次
ERM：MovieShowTime 時間衝突規則

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 無間隔：場次可以緊接在前一場結束後立即開始 |
| B | 固定間隔：場次之間需要固定間隔（如 15 分鐘清場時間） |
| C | 可配置：間隔時間可依影廳類型配置（如 IMAX 需較長清場時間） |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

- 設定場次的時間衝突檢查邏輯
- 時刻表 UI 的場次顯示
- MovieShowTime 的結束時間計算

# 優先級

Medium
- 影響場次排程的合理性，但可延後決定

---
# 解決記錄

- **回答**：A - 無間隔：場次可以緊接在前一場結束後立即開始，由管理者自行決定是否預留間隔
- **更新的規格檔**：無需更新（現有規格已符合此行為，只檢查時間重疊）
- **變更內容**：確認場次間無強制間隔要求，系統只檢查時間段不重疊
