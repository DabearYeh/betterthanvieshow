# 刪除電影_若電影有已排程的場次是否可以刪除

刪除電影功能中，若電影有已排程的場次,是否可以刪除該電影？

# 定義

Feature：刪除電影 feature，關於「電影有已排程的場次時是否可以刪除」

# 多選項

| 選項 | 描述 |
|--------|-------------|
| A | 不可以刪除，必須先刪除相關場次 |
| B | 可以刪除，自動刪除相關場次 |
| C | 可以刪除，但場次狀態改為已取消 |

# 影響範圍

- 刪除電影.feature 中的刪除限制規則
- 資料級聯刪除的設計
- 業務規則的完整性

# 優先級

High - 影響資料安全與業務邏輯

# 解決記錄

- **決議**：A  不可以刪除，必須先刪除相關場次
- **更新規格檔案**：spec/features/刪除電影.feature
- **變更內容**：新增規則「若電影有已排程的場次則無法刪除電影」
