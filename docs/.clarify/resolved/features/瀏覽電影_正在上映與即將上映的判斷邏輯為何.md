# 釐清問題

瀏覽電影功能中，正在上映與即將上映的判斷邏輯為何？

# 定位

Feature：瀏覽電影功能
Rule：可以瀏覽正在上映的電影、可以瀏覽即將上映的電影

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 根據 Movie 的狀態欄位判斷 |
| B | 根據場次日期與當前日期比較判斷 |
| C | 根據 Movie 的上映日期欄位判斷 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

- Movie 實體的屬性定義
- 瀏覽電影功能的規則邏輯
- 電影篩選與分類邏輯
- 測試案例設計

# 優先級

High
- 影響電影上映狀態的判斷方式
- 影響 Movie 實體設計（是否需要狀態或日期欄位）

---

# 解決記錄

**使用者選擇：B**

**規格文件變更：**
- `spec/features/瀏覽電影.feature` - 更新規則說明：
  - Rule「可以瀏覽正在上映的電影」：明確判斷邏輯為電影有場次且場次日期 >= 當前日期
  - Rule「可以瀏覽即將上映的電影」：明確判斷邏輯為電影有場次且所有場次日期 > 當前日期

**說明：**
不需要 Movie 實體增加新的日期欄位，直接透過查詢 MovieShowTime 的 show_date 來判斷
