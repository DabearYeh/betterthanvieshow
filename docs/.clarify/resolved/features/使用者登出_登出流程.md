# 使用者登出流程

使用者登出需要做什麼？

# 定義

Feature：使用者登入登出系統 > 使用者登出流程

# 多選項

| 選項 | 描述 |
|--------|-------------
| A | 清除登出時，清除 session，重導向首頁 |
| B | 清除登出時，清除 session，重導向登入頁 |
| C | 清除登出時，清除 session 與 token，重導向首頁 |
| D | 其他（請說明） |

# 影響範圍

- 登出功能的前後置條件
- Session/Token 管理策略
- 登出後的頁面導向

# 優先級

Medium
- 影響登出功能的完整性

# 解決記錄

- **決議**：B — 清除 session + 重導向登入頁
- **理由**：登出時自動導向登入頁讓使用者的下一步操作更清晰，體驗較即時友善
- **更新規格檔案**：spec/features/使用者登出.feature
