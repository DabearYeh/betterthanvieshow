# 釐清問題

驗票功能中，如何判斷票券是否過期？

# 定位

Feature：驗票功能
Rule：必須檢查票券是否過期、過期的票券不可使用

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 檢查場次日期與時間是否已過 |
| B | 檢查 Ticket 是否有過期時間欄位 |
| C | 檢查場次結束時間是否已過 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

- Ticket 或 MovieShowTime 的屬性定義
- 驗票功能的前置條件驗證
- 票券過期判斷邏輯
- 測試案例設計

# 優先級

High
- 影響驗票核心邏輯
- 影響票券過期的判斷方式

# 解決記錄

**使用者選擇：C**

**規格文件變更：**
- `spec/features/驗票.feature` - 更新規則說明：
  - Rule「必須檢查票券是否過期」：明確判斷邏輯為檢查該票券所屬場次的結束時間 (MovieShowTime.end_time) 是否已過，若當前時間 > 場次結束時間則視為過期
