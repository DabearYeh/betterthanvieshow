# 訂票_是否有訂票數量限制

訂票功能中,是否有訂票數量的限制？

# 定義

Feature：訂票功能
Rule：必須選擇座位

# 多選項

| 選項 | 描述 |
|--------|-------------
| A | 有限制,每次最多訂購 N 張票 |
| B | 無限制,只要座位足夠即可 |
| C | 依場次或影廳設定不同限制 |
| Short | 其他（請簡短答案<=5 字）|

# 影響範圍

- 訂票功能的前置條件邏輯
- 座位選擇的 UI 限制
- 訂票規則設計

# 優先級

Medium
- 影響訂票業務規則
- 影響使用者體驗

---

# 解決記錄

- **決議**：A  有限制,每次最多訂購 6 張票
- **依據**：spec/erm.dbml Order 表 ticket_count 欄位：「每張 Order 最多可訂 6 張票」
- **更新規格檔案**：無需修改（已在 ERM.dbml 中定義）
