# 釐清問題

Cinema 是否需要記錄地址、聯絡資訊等基本資料？

# 定位

ERM：Cinema 實體，可能需要新增 address、phone、email 等屬性

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 需要，包含地址、電話、email 等 |
| B | 不需要，僅需 id 與 name |
| C | 需要地址，但不需要聯絡資訊 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

- Cinema 實體的屬性定義
- 影城管理功能（若有）
- 前台顯示邏輯（若需顯示影城資訊）

# 優先級

Low
- 對核心功能影響較小
- 可在後續階段補充
