# 釐清問題

Movie 是否需要上映狀態欄位來區分正在上映與即將上映的電影？

# 定位

ERM：Movie 實體，可能需要新增 release_status 或類似屬性

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 需要狀態欄位（如：未上映、上映中、已下檔） |
| B | 不需要狀態欄位，透過場次日期判斷 |
| C | 需要上映日期欄位，由日期決定狀態 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

- Movie 實體的屬性定義
- 瀏覽電影功能的規則（正在上映/即將上映的區分邏輯）
- 電影管理功能的規則

# 優先級

High
- 影響核心業務邏輯（正在上映/即將上映的判斷）
- 影響瀏覽電影功能的實作

---
# 解決記錄

- **回答**：A - 需要狀態欄位（如：未上映、上映中、已下檔）
- **更新的規格檔**：spec/erm.dbml
- **變更內容**：在 Movie 實體中新增 release_status 屬性（上映狀態），明確定義所有可能的狀態值
