# 釐清問題

編輯影廳時，若增加座位總數，新增座位的自動編號規則是什麼？如何分配排號與座號？

# 定位

ERM：Theater.total_seats, Seat.row_name, Seat.seat_number
Feature：編輯影廳 → Rule: 修改座位總數時需同步調整座位配置 → Example: 增加座位時自動建立

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 依照現有排數順序，在最後一排繼續編號（如已有 A1-A50，新增則為 A51-A60） |
| B | 新增座位後自動添加新排（如已有 A1-A50，新增則為 B1-B10） |
| C | 根據行列網格自動填充（先填完現有排，再新增排） |
| D | 完全由管理者手動指定 |
| Short | 其他規則 |

# 影響範圍

- Theater 座位總數與 Seat 記錄的同步邏輯
- 編輯影廳功能的實現方式
- 座位配置設定功能的依賴關係
