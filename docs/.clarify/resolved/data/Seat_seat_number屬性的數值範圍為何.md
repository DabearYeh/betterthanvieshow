# 釐清問題

Seat 的 seat_number 屬性的數值範圍為何？

# 定位

ERM：Seat 實體的 seat_number 屬性

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | seat_number >= 1（正整數） |
| B | seat_number > 0（正整數，可從任意數字開始） |
| C | seat_number >= 0（包含 0） |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

- Seat 實體的屬性定義
- 設定座位配置功能的前置條件與驗證規則
- 座位編號的邊界測試案例

# 解決記錄

- **回答**：A - seat_number >= 1（正整數，一般從 1 開始）
- **更新的規格檔**：spec/erm.dbml
- **變更內容**：Seat 實體 seat_number 屬性已定義為正整數從 1 開始，無需修改
