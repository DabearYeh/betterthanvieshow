# 釐清問題

MovieShowTime 是否需要記錄場次的結束時間？

# 定位

ERM：MovieShowTime 實體，可能需要新增 end_time 或類似屬性

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 需要明確的結束時間欄位 |
| B | 不需要，由開始時間加上電影時長計算 |
| C | 需要結束時間，以處理中場休息等情況 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

- MovieShowTime 實體的屬性定義
- 場次衝突檢查邏輯（同一影廳的時段重疊判斷）
- 設定場次功能的驗證規則

# 優先級

High
- 影響場次衝突檢查的核心邏輯
- 影響同一影廳多場次的排程設計

# 解決記錄

- **回答**：C - 需要結束時間，以處理中場休息等情況
- **更新的規格檔**：spec/erm.dbml
- **變更內容**：在 MovieShowTime 實體中新增 end_time 屬性（結束時間），用於記錄包含中場休息等時間的實際結束時間
